# Comunidad

## Versión

Rsbuild sigue la especificación de Versionado Semántico.

Para más detalles, consulta [Releases](/community/releases/index).

El desarrollo de Rsbuild es impulsado por el equipo de Rspack de ByteDance y los contribuyentes de la comunidad.

Para obtener información sobre los miembros del equipo, consulta [Rspack - Core Team](https://www.rspack.dev/misc/team/core-team).
