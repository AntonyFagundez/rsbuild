# Rsbuild Types

Esta sección describe algunas de las definiciones de tipo proporcionadas por Rsbuild.

## RsbuildInstance

El tipo de instancia de Rsbuild, correspondiente al valor de retorno del método [createRsbuild](/api/javascript-api/core#creatersbuild).

```ts
import type { RsbuildInstance } from '@rsbuild/core';

let rsbuild: RsbuildInstance;
```

## RsbuildConfig

El tipo de configuración de Rsbuild.

```ts
import type { RsbuildConfig } from '@rsbuild/core';

const config: RsbuildConfig = {
  // ...
};
```

También puedes importar las definiciones de tipo de cada campo en la configuración de Rsbuild:

```ts
import type {
  DevConfig,
  HtmlConfig,
  ToolsConfig,
  SourceConfig,
  ServerConfig,
  OutputConfig,
  SecurityConfig,
  PerformanceConfig,
  ModuleFederationConfig,
} from '@rsbuild/core';
```

## NormalizedConfig

El tipo de configuración de Rsbuild después de la normalización, correspondiente al valor de retorno del método [getNormalizedConfig](/plugins/dev/core#apigetnormalizedconfig).

```ts
import type { NormalizedConfig } from '@rsbuild/core';

const config: NormalizedConfig = api.getNormalizedConfig();
```

También puedes importar las definiciones de tipo de cada campo en la configuración normalizada de Rsbuild:

```ts
import type {
  NormalizedDevConfig,
  NormalizedHtmlConfig,
  NormalizedToolsConfig,
  NormalizedSourceConfig,
  NormalizedServerConfig,
  NormalizedOutputConfig,
  NormalizedSecurityConfig,
  NormalizedPerformanceConfig,
  NormalizedModuleFederationConfig,
} from '@rsbuild/core';
```

## NormalizedEnvironmentConfig

El tipo de configuración de entorno de Rsbuild después de la normalización, correspondiente al valor de retorno del método [`getNormalizedConfig({ environment })`](/plugins/dev/core#apigetnormalizedconfig).

```ts
import type { NormalizedEnvironmentConfig } from '@rsbuild/core';

const config: NormalizedEnvironmentConfig = api.getNormalizedConfig({
  environment,
});
```

## RsbuildContext

El tipo de la propiedad [context](/api/javascript-api/instance#rsbuildcontext) en la instancia de Rsbuild.

```ts
import type { RsbuildContext } from '@rsbuild/core';

const context: RsbuildContext = rsbuild.context;
```

## RsbuildPlugin

El tipo de plugin de Rsbuild.

```ts
import type { RsbuildPlugin } from '@rsbuild/core';

const myPlugin: RsbuildPlugin = {
  name: 'my-plugin',
  setup() {},
};
```

## RsbuildPluginAPI

El tipo del objeto `api` pasado a la función `setup` en el plugin de Rsbuild.

```ts
import type { RsbuildPluginAPI } from '@rsbuild/core';

const myPlugin = {
  name: 'my-plugin',
  setup(api: RsbuildPluginAPI) {},
};
```

## RsbuildTarget

El tipo de destino de compilación.

```ts
import type { RsbuildTarget } from '@rsbuild/core';
```

## CreateRsbuildOptions

El tipo de parámetro del método [createRsbuild](/api/javascript-api/core#creatersbuild).

```ts
import type { CreateRsbuildOptions } from '@rsbuild/core';
```

## InspectConfigOptions

El tipo de parámetro del método [rsbuild.inspectConfig](/api/javascript-api/instance#rsbuildinspectconfig).

```ts
import type { InspectConfigOptions } from '@rsbuild/core';
```

## Rspack

Incluye todos los tipos exportados por `@rspack/core`, como `Rspack.Configuration`.

```ts
import type { Rspack } from '@rsbuild/core';

const rspackConfig: Rspack.Configuration = {};
```
